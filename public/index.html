FAQs

How do I switch between light and dark mode?

In Settings → Theme (the first card), use the toggle:

  • Auto — follows your device/OS theme (default)
  • Light — forces light mode
  • Dark — forces dark mode

Your choice is saved locally and persists across sessions.

What do the Tracker and Reports tabs do?

Tracker shows your current month's expenses up to today. You can add new expenses, search within the current month, and see a summary breakdown by category.

Reports gives you the full picture — filter by year, month, category, or search across all data. It also shows a 12-month Category Breakdown chart that you can toggle between Amounts and Percentages view.

How do I add an expense?

On the Tracker tab, fill in the Add Expense form:

  • Date — defaults to today. Click "Today" to reset.
  • Details — describe the expense. Smart Suggestions will auto-fill the category and amount if you've entered this item before.
  • Category — select or change the suggested category.
  • Amount — enter the amount (decimals allowed).
  • Add note — optional note for this expense.

Click + Add to save. The app checks for duplicates and will warn you if a similar entry already exists.

What is the Smart Suggestions feature?

When you type a details name you've used before (e.g. "Milk"), the app automatically fills in the category and amount from your last entry with those details. This saves time on recurring expenses.

The autocomplete dropdown shows matching history as you type (minimum 2 characters). You can click or use arrow keys + Enter to select.

How do I enable Abroad Mode?

In Settings → Expenses Abroad, click Abroad in the toggle, then select the foreign currency from the dropdown. It auto-saves immediately. When enabled:

  • The amount label changes to show the foreign currency symbol (e.g. $ instead of ₹).
  • Enter amounts in the foreign currency — they are auto-converted to your base currency using the exchange rate you defined.
  • The original amount and currency are stored and displayed alongside the converted amount in the table.

Click Home to deactivate. You need to define the exchange rate first in Settings → Currency Rates.

How do I copy an expense to future months?

Click the Copy button on any expense row. A modal opens where you can:

  • Set the current occurrence date (the date of the original expense).
  • Enter the number of months to copy forward (max 60).

The app generates dates starting from the next month, clamping to the last day if needed (e.g. copying from Jan 31 to Feb → Feb 28). Months that already have this expense are skipped. A notification is created when a recurring series is approaching its end.

How do I edit or delete an expense?

Each expense row has action buttons:

  • Edit — opens a modal to change date, details, category, amount, or note.
  • Delete — removes the expense after confirmation.
  • Notes — view or add a note to the expense.

These work both on the Tracker tab and the Reports tab.

How do I use the Reports filters and search?

The Reports tab has several filtering options:

  • Search — type to filter by expense details. Click All to search across all data (ignoring year/month).
  • Year / Month — select a specific period. When both are set, day-of-month filter buttons appear below.
  • Category — filter by a specific category.
  • Day buttons — click a day to see only that day's expenses. Shift+click to select a range.

Click Reset to clear all filters. Use ↓ CSV to download filtered results.

How do I batch-update expenses in Reports?

In the Reports tab, check the boxes next to expenses (or use the Select All checkbox in the header). A batch bar appears with options to:

  • Change Category — select a new category and apply it to all selected expenses.
  • Rename Selected — enter new details text to rename all selected expenses at once.

Click Clear to deselect all.

What is the Cash Flow Forecast tab?

The Forecast tab lets you project your future monthly balances. You can:

  • Set a Starting Balance by clicking the first month's cell.
  • Add Income rows (e.g. Salary) and Expense rows (e.g. Rent, Groceries).
  • Click any cell to edit the amount for that month.
  • Drag the handle to reorder rows.
  • Right-click or long-press a cell to add a note.
  • Click a row label to rename it, or the ✕ button to remove it.

Use the Apply button to set the start month and number of months (1–24). ↓ CSV downloads the forecast.

How do I set up currency and exchange rates?

In Settings:

  1. Base Currency — select your home currency (e.g. INR, USD, EUR). All expenses are stored in this currency.
  2. Currency Rates — define exchange rates for foreign currencies (e.g. 1 USD = 83 INR). These are used by Abroad Mode to auto-convert expenses.

You can add, edit, or delete currency rates as needed.

How do I manage categories?

In Settings → Categories:

  • Add — click "+ Add Category", enter a name and pick a color.
  • Reorder — use the ↑ and ↓ buttons to change the order.
  • Change color — click the color dot next to a category name.
  • Rename — click the edit button, edit the name inline, and save.
  • Delete — removes the category permanently. Expenses using it will lose their category assignment, so consider renaming instead.

Max 15 categories. Changes apply immediately to all existing expenses.

What are the Notifications for?

The Notification Bell in the header shows reminders for recurring expenses. When a copied expense series (created via the Copy button) is approaching its last occurrence, a notification appears to let you know it's ending soon.

Click the bell to view notifications. Each one can be dismissed (removed from the list) or permanently deleted.

What is the Quick Notes scratchpad for?

The Quick Notes button (📄 icon in the header) opens a scratchpad where you can jot down anything — budget goals, reminders, things to track. Notes are saved automatically when you click the Save button.

This is a simple text area with no formatting. Perfect for quick thoughts while managing your expenses.
